为什么 MCP tools 比官网接口多

Tushare MCP tools 数量是 258，官网当前左侧文档树整理出的接口/数据服务页是 237。这个差额不能简单理解成“多 21 个真实接口”，更准确地说是两个口径互相错位：MCP 里有历史接口、隐藏接口、特殊服务和自选股操作类工具；官网当前目录树里也有部分新接口没有被 MCP 暴露。
